    Tips for Step 1: Event Details

    Here are some expert tips for setting up your event details:

    • Make sure your event's timezone is correct.
    • Add FAQs to outline event logistics (refund policy, parking, dress code, etc.)
    • Add color to your event description with video or photos.
    • Spruce up your organizer profile page to promote your events.
    • Learn about setting up repeating events with ease.

    Need more help? Check out our Help Center for more information.

    Tips for Step 2: Create Tickets

    Eventbrite is 100% free to use if you're organizing a free event. For paid tickets, we charge a small fee, which you can pass on to attendees, or pay yourself. Check out some more quick tips about setting up tickets and registrations:

    • Create a seat map and set up reserved seating
    • Find out how to fundraise and collect donations
    • See if you qualify for our nonprofit fee discount
    • Set up group discounts
    • Cap your event's capacity to make sure you don't oversell
    • Run an event without paper tickets

    Need more help? Check out our Help Center for more information.

    Tips for Step 3: Additional Settings

    There are four levels of privacy for events that you can adjust at any time:

    • Public: Your event is visible on our site and search engines like Google. With public events, pick an event type and topic so attendees can find your event.
    • Private: Your event can't be found via search, but people with the URL can see the event page. (You can choose to let the event be shared on social media.)
    • Password protected: You set a password attendees must enter to see your event page.
    • Invite-only: Attendees can only register through the Eventbrite invitation you send them.

    Want to spread the word? Here are free tools you can use through Eventbrite:

    • Publish your event to Facebook
    • Send out email invitations
    • Customize your event page URL
    • Create and send discount codes
    • Sell tickets right on your website with Eventbrite widgets
    • Pull in tweets and photos onto your event page

    Need more help? Check out our Help Center for more information.
